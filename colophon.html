On Using AI

I want to be upfront: the code for this site was written with the help of Claude, an AI made by Anthropic. I'm not a web developer — I'm a physics student. I wanted a personal website but I didn't know HTML or CSS, so I used AI to help me build it.

That said, this wasn't a case of "make me a website" and walking away. The design, the structure, and what the site feels like came out of a long conversation where I pushed back on things I didn't like, asked for changes, and tried to make something that actually felt like me. Every piece of writing on this site is mine. The code was a collaboration.

I think there's a version of using AI tools that's dishonest — passing off generated work as entirely your own, or using it to fake competence you don't have. I'm trying to do the opposite. I'm learning HTML and CSS as I go, reading the code, and making changes myself. The goal is to eventually understand every line on this site. I'm not there yet, but I'm working on it.

I'd rather be transparent about this than pretend I hand-coded everything from scratch.

Details

Built with: HTML, CSS, and a text editor. No frameworks, no JavaScript, no build tools.

Fonts: Newsreader, Space Mono, and Inter — all free from Google Fonts.

Hosted on: GitHub Pages.

AI assistance: Claude (Anthropic) helped write the HTML and CSS. All written content is my own.
